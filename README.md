# Nexter-web-page
Advanced web-page focusing on an application of CSS Grid. Based on
Udemy's Advanced CSS and Sass: Flexbox, Grid, Animations and More!
course.

## Technology stack: 
Advanced CSS, Sass, RWD.

## Live demo:
https://achazu.github.io/Nexter-web-page/
